# 🧩 15 Puzzle Game (C Language)

Hi! 👋  
This is my 15 Puzzle Game made using C language.

It is a console-based game where you arrange numbers from 1 to 15 in order using arrow keys.

---

## 🎮 How to Play

- Use Arrow Keys to move tiles
- Arrange numbers from 1 to 15
- Keep empty space at bottom-right
- Press E to exit

---

## 💻 How to Run

Compile using GCC:

gcc game.c -o game
game

---

## 📚 Concepts Used

- 2D Arrays
- Functions
- Pointers
- Random number generation
- Arrow key handling
